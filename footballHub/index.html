Football Hub

I created Football Hub to gain a better understanding of how to consume an API. I'm not a fan of football myself, but my Step-Son is football mad, so I thought it would give enough complexity for me to practice, whilst the result could be of benefit. Please be aware that Football Hub is still very much a work in progress.
All data is courtesy of football-data.org, who provide a free public API that allows access to the data required to create this application. If you're looking for a good resource to practice their own web skills, then I highly recommend this site.
